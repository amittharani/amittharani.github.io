Marbellous game



Marbellous is casual puzzle game where the player interacts with marbles on a restricted tile platform. The goal is to align similar colored marbles in minimum moves possible to attain a higher score.

Game Abstract
Design a windows application with following attributes:
1. Render balls, defined by color but sharing common properties
2. A tile based platform for the movement of balls
3. Rules for ball to ball and ball to boundary collision
4. Game loop script to accept direction inputs and move balls
5. Moves count and scoring mechanism
6. Load maps for different levels

Programming
The game was programmed using C++ language. I used SFML (Simple and Fast Multimedia Library) for windows, to accept user inputs, render 2D graphics (textures, sprites and text) and integrate music.
Microsoft Visual Studio was the choice of IDE/code editor for the game development.

Marble class properties: Postition, marble-type, sprite(image)and direction.
Loading Map: A 2D matrix of 1s and 0s is used for permeable and non-permeable tiles.
Loading Marbles: A text file with #marbles, moves count for each rating, and a 2D matrix with co-ordinates & marble-type for all marbles.
